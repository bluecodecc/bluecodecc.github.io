Java之UML类图 2019-04-07

前言

之前学习Java阅读《疯狂的Java讲义》的时候就看过关于UML相关的内容，工作后需要了解整个项目的架构的时候就想过画UML，想用起来的时候就忘了，上次看过也没做记录，现在需要重新梳理新项目的时候，却又忘了，所以这次总结下吧。

类图

类在类图中使用包含三个部分的矩形来表述，最上面的部分显示类的名称，中间部分包含类的属性，最下面的部分包含类的方法,

类之间有三种关系

关联（包括聚合、组合）

关联具有一定的方向性，如果仅能从一个类单方向访问另一个类，则被称为单向关联；如果可以互相访问，则被称为双向关联一个对象能访问关联对象的数目被称为多重性。关联使用一条实线来表示，带箭头的实线表示单向关联。
