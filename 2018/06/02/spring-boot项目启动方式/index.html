spring-boot学习 2018-06-02

spring-boot启动方式

  • 直接IDE启动Main方法
  • 进入项目根目录，运行mvn spring-boot:run
  • 进入项目根目录运行mvn install后进入target目录会看到生成的jar文件,执行java -jar xxx.jar运行

    配置文件切换

  • 可以建立多个配置文件，例如application-dev.yaml, application-prod.yaml和application.yaml等配置文件, 在application.yaml配置文件中进行配置,

  • 当用mvn install编译,使用java -jar xxx.jar运行时指定配置java -jar xxx.jar --spring.profiles.avtive=dev

组合注解

  • @RestController相当于@Controller @ResponseBody
    当一个类用@Controller注解时，只能返回一个对应的页面，加上@ResponseBody后就可以返回指定的内容了。相当于@RestController注解
  • @RequestMapping(value = "hello",method = RequestMethod.GET)等注解可以有一种简写形式@GetMapping(value = "hello")

如何接收参数

  • 路径

  • 参数

❗️可能存在不能取到值的情况，这样就会报错，可以提供一个默认值
